BoardScribe

BoardScribe - Online learning Platform with Image Recognition

The goal of BoardScribe is to allow students to take a picture of their notes and upload them to a platform that converts the picture into markup that can be viewed by all members of a class.

This allows students to collaborate on school notes and have discussion on challenging topics. Using image recognition, all students have to do is upload a picture of the blackboard and BoardScribe does the rest.

Built at WildHacks 2016

Technologies used

Clarifai Image Recognition API

Tesseract.js

Django

PostgreSQL

Next Steps and Links

Some things we would like to add include improving the image recogonition, adding support for the markup of equations and formulas and also building a Youtube reccomendation system to suggest videos related to the topic of the note.

BlazeSMS

Text Messaging Service for scheduling Doctor's Appointments

The goal of BlazeSMS is to allow patients to schedule Doctor's appointments with one simple text message, no apps or phone call required.

Patients simply text a specified number their appointment request which includes details like the date and their reason for scheduling the appointment.

Using the Twilio API that message hits our server and the patient recieves a reply either confirming their request or suggesting an alternate time if that spot is booked. We also built a simple dashboard for administrators of a Doctor's office/Hospital

Built at HackWestern 2016

Technologies used

Bootstrap

Node.js

Express

MongoDB

Twilio

Next Steps and Links

Some things we would like to add include improving the responses of the service using AI like a chatbot, adding support for multiple doctor's offices and expanding to other industries that use appointments

Arduino-Blackjack

Dice Jack

Project Description


As part of our final project for our Digital Systems Course(SYDE 192L), we made an Arduino version of the classic game Blackjack. Our project had to include hardware interrupts with at least 2 interrupts per interrupt pin and Timers

There are 2 players in the game and on each turn they have 2 options. Either they can press the hit button or the stay button and they alternate turns until someone goes over 21 or there are 7 turns by each player. We also did an animation with LED lights to simulate a dice roll.
