BlazeSMS

BlazeSMS

The goal of BlazeSMS is to allow patients to schedule Doctor's appointments with one simple text message, no apps or phone call required.

Patients simply text a specified number their appointment request which includes details like the date and their reason for scheduling the appointment.

Using the Twilio API that message hits our server and the patient recieves a reply either confirming their request or suggesting an alternate time if that spot is booked. We also built a simple dashboard for administrators of a Doctor's office/Hospital

Built at HackWestern 2016

Arduino
Arduino-Blackjack

Dice Jack

Project Description


As part of our final project for our Digital Systems Course (SYDE 192L), we made an Arduino version of the classic game Blackjack. Our project had to include hardware interrupts with at least 2 interrupts per interrupt pin and Timers

There are 2 players in the game and on each turn they have 2 options. Either they can press the hit button or the stay button and they alternate turns until someone goes over 21 or there are 7 turns by each player. We also did an animation with LED lights to simulate a dice roll.

Project Description

FeelsLike is an app that allows Users to see the temperature in their city with a brief description of the conditions as well as see the 'Feels Like' temperature with a windchill calculator.
